Developed a Java-based preliminary wastewater pump sizing calculator that automates flow estimation and hydraulic screening 
for commercial facilities. The tool integrates EPA Table 3-4 design standards and ASME B36.10M pipe specifications to rapidly
translate facility occupancy data into actionable pumping parameters, including daily flow rates, metric conversions, and 
static head requirements. Designed for engineering screening workflows, it streamlines early-stage infrastructure planning by 
codifying regulatory guidelines into a repeatable computational format. This project demonstrates proficiency in applying fluid 
mechanics fundamentals, regulatory compliance mapping,and software automation to optimize preliminary mechanical design processes.
